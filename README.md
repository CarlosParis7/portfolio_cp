## 📌 Overview
This repository houses the source code for my personal and professional web presence. Unlike a traditional static CV, this site serves as a **dynamic product portfolio** designed to demonstrate UI/UX rigor, technical architecture, and a strong focus on business outcomes (Logistics, AI, and Automation).
## ⚠️ The Goal
To establish a centralized digital footprint that communicates *value* rather than just *tech stacks*. It acts as the primary top-of-funnel entry point for potential clients, technical recruiters, and strategic partners.
## 💡 The Solution
A highly optimized web application built from the ground up prioritizing speed, accessibility, and clean visual hierarchy. The architecture allows for easy updates to case studies, fast loading times for mobile users, and a premium aesthetic that aligns with a high-level "Product Engineer" positioning.
## ✨ Core Features
- **⚡ Blazing Fast Performance**: Statically generated (SSG) / Server-Side Rendered (SSR) for optimal Lighthouse scores and SEO.
- **📱 Mobile-First Responsive Design**: Flawless execution across all device sizes using modern CSS utility classes.
- **🎨 Premium UI/UX Aesthetic**: Clean, distraction-free interface emphasizing typography, whitespace, and micro-interactions.
- **🔗 Seamless Integrations**: Direct links to live SaaS products, n8n webhook triggers for the contact form, and embedded external assets.
## 🛠️ Tech Stack & Architecture
- **Framework**: [Next.js (App Router) / React] *(Ajusta si usas Vue/Vite, etc.)*
- **Styling**: [TailwindCSS / Framer Motion para animaciones]
- **Deployment & Hosting**: [Vercel / Netlify]
- **Content/Data**: [Markdown / Supabase / Hardcoded JSON]

## ⚙️ Local Development
To run this personal hub locally:
```bash
# Clone the repository
git clone https://github.com/tu-usuario/founder-portfolio.git
# Navigate into the directory
cd founder-portfolio
# Install dependencies
npm install
# Start the development server
npm run dev
```
